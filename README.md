================================================================================
                            项目简介
================================================================================

PixelBee 是一款企业级拼豆图纸生成工具，由一位八年级学生独立开发。它可以：

  📤 上传任意图片（JPG / PNG）
  🎨 自动匹配最接近的拼豆色号（支持 H系列 / Artkal 双色板）
  📐 自定义图纸尺寸（16x16 ~ 1000x1000，支持锁定图片比例）
  🔍 颜色比较器（显示原色→匹配色 + 相似度百分比）
  ✏️ 手动修改任意格子的颜色（点击选中 + 按钮换色）
  ⭐ 特殊色过滤（荧光、金属、夜光等可单独禁用）
  💾 文件管理（保存 / 加载 / 删除图纸，浏览器本地存储）
  📷 导出高清 PNG 图纸
  📊 导出 CSZ 格式（可用 Excel / WPS 打开编辑）
  ✋ 抓手工具 + 缩放（超大图纸随意浏览）

================================================================================
                      ⭐ 色号数据输入方法（重要！）
================================================================================

  由于我没有时间把所有品牌的颜色都录进去，所以你拿到项目后，需要自己动手
  把拼豆色号填进 palettes.js。别担心，很简单！


  📝 第一步：找到 js/palettes.js 文件

  用记事本或 VS Code 打开 js/palettes.js，你会看到类似这样的代码：

  "h_series": {
      name: "H系列 (国产通用)",
      colors: [
          { id: "H01", hex: "#F8F9FA" },
          { id: "H02", hex: "#FFFFFF" },
          // ... 更多颜色
      ]
  }


  📝 第二步：按格式添加颜色

  每个颜色对象由三部分组成：
    - id    ：色号（如 "H01"、"S01"、"A01"）
    - hex   ：十六进制颜色代码（如 "#FF0000" 代表红色）
    - special（可选）：如果是特殊色（荧光、金属、夜光等），加上 special: true

  示例：
    { id: "H21", hex: "#F4D03F", special: true },   // 金色（特殊色）
    { id: "H22", hex: "#BDC3C7", special: true },   // 银色（特殊色）


  📝 第三步：如何获取正确的十六进制代码？

  1. 用手机或相机拍一张拼豆色卡的照片（要清晰、光线均匀）。
  2. 打开在线取色工具，例如：https://imagecolorpicker.com
  3. 上传照片，鼠标移到某个颜色上，就能看到它的十六进制代码（如 #E63946）。
  4. 把代码填到 palettes.js 对应的色号后面。


  📝 第四步：新增品牌

  如果你想添加一个新品牌（比如 MARD、COCO 等），按这个格式在 PALETTES
  对象中添加：

  "mard_series": {
      name: "MARD 系列",
      colors: [
          { id: "M01", hex: "#FFFFFF" },
          { id: "M02", hex: "#1A1A1A" },
          // ... 更多颜色
      ]
  }

  保存文件后，刷新 tool.html，下拉菜单中就会自动出现你添加的品牌！


================================================================================
                            使用方法
================================================================================

  1️⃣ 上传图片
     - 点击右侧的"点击上传"区域，或直接把图片拖进去。
     - 支持 JPG 和 PNG 格式。

  2️⃣ 设置参数
     - 拼豆品牌：选择你手中拥有的豆子品牌（H系列 / Artkal）。
     - 图纸尺寸：选择板子大小（16x16 / 29x29 / 52x52 / 自定义）。
     - 豆子大小：每个格子的像素大小（8~60px），固定不变，图纸整体会变大/变小。
     - 限制最大颜色数：控制图纸使用的颜色数量（不限 / 4色 / 8色 / 16色 / 自定义）。
     - 特殊色开关：取消勾选后，所有标记为 special: true 的色号不会被匹配。

  3️⃣ 生成图纸
     - 点击 "⚡ 生成拼豆图纸"，等待进度条走完。
     - 生成后，右侧会显示：
       - 颜色比较器：每个颜色的原色→匹配色 + 相似度。
       - 颜色图例：使用的色号及数量（特殊色带 ⭐ 标记）。

  4️⃣ 修改格子颜色（选色模式）
     - 在 "选色" 模式下，点击任意格子 → 该格子被高亮（金色边框）。
     - 点击 "更换颜色" 按钮 → 弹出颜色选择器。
     - 点击一个色号 → 点击 "确认" → 颜色生效。

  5️⃣ 浏览大图纸（抓手模式）
     - 切换到 "抓手" 模式，可以拖动画布平移。
     - 用 "缩放" 滑块放大/缩小视图。

  6️⃣ 保存与导出
     - 保存：输入图纸名称，点击"保存" → 存储在浏览器本地。
     - 加载：从下拉列表中选择已保存的图纸，点击"加载"。
     - 删除：选中图纸，点击"删除"。
     - 导出 PNG：导出高清图片。
     - 导出 CSZ：导出类 CSV 格式，可用 Excel / WPS 打开编辑。


================================================================================
                      ⚠️ 已知问题（请知悉）
================================================================================

  由于本项目是一个八年级学生在有限时间内开发的，可能存在一些小问题，
  主要包括：

  1. 色号数据不完整：我没有时间把所有品牌的颜色都录进去，所以你需要
     按照上面的方法自己添加色号。

  2. 超大图纸性能：当网格数超过 200x200 时，渲染可能会变慢或卡顿。
     建议使用 100x100 以内的尺寸。

  3. 浏览器兼容性：推荐使用 Chrome 或 Edge 浏览器。Safari 和 Firefox
     可能在某些功能上有兼容问题。

  4. 换色功能：目前需要"点击选中 → 点击更换颜色按钮 → 选择色号 →
     点击确认"四步操作，稍显繁琐，但稳定可靠。

  5. 特殊色过滤：需要手动在 palettes.js 中为每个特殊色添加 special: true
     标记。

  如果你在使用过程中发现其他问题，欢迎通过邮箱联系我！我会尽力修复 🛠️
